About the Tool

This tool helps you organize files into different folders if you have some messy files inside your folder and want to organize them.

For now, it works just to organize files for my 1st semester subjects.

It uses regex to find the terms from file names that match with each subject category and assigns that file to the particular subject.

How It Works

Messy files in subjects_notes directory

Terminal command to run our tool

Rearranged messy files in their respective subdirectory
